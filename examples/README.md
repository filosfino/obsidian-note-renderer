# Examples

These notes are meant to be copied into an Obsidian vault and rendered with Note Renderer.

Start with:

- `01-basic-note.md` — the smallest useful structure: title, cover text, and body pages.
- `02-rich-cover-and-images.md` — cover styling syntax plus body image embed modifiers.
- `03-preset-frontmatter.md` — lightweight `renderer_config` frontmatter using a preset name plus local overrides.

Image examples use placeholder filenames such as `cover-demo.jpg` and `diagram.png`. Replace them with files that exist in your vault.
